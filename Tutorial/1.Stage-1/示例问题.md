## 示例问题

问题 - 选购阶段（服饰鞋帽行业）
### A. 按功效/功能需求搜索
```
显瘦的连衣裙 / 身材微胖穿什么连衣裙？
通勤西装套装 / 想要看起来干练又不老气？
防风保暖的羽绒服真的暖吗？
让腿显长的裤子 / 小个子穿什么裤子？
修复娇嫩肌肤的儿童外套 / 宝宝皮肤敏感穿什么？
能防滑耐磨的运动鞋？/ 跑步时脚打滑怎么办？
染发后穿什么颜色衣服更衬肤色？
久坐不皱的西装裤 / 上班坐一天裤子还平整吗？
```
### B. 按身材/肤质/使用场景搜索
```
梨形身材 / 臀宽腿粗穿什么裤子？
干性皮肤 / 秋冬穿什么内搭不扎？
白皙肤色穿什么颜色显气质？
敏感肌/易过敏体质，衣服材质要注意什么？
```
### C. 按成分/材质/人群偏好搜索
```
纯棉衣服真的好吗？会不会容易皱？
亚麻衣服凉快吗？适合夏天吗？
含羊毛/羊绒的衣服会扎吗？
孕妇可用的安全孕妇装？
儿童/宝宝衣服推荐？
```
### D. 价格/品牌/促销场景需求
```
平价好穿的通勤装有推荐吗？
高端/设计师品牌衣服值不值？
[XX品牌]衣服怎么样？求真实评价？
刚推荐的这款波司登羽绒服，之前有过更低价吗？
这款有更低价时，能提醒我吗？
```
### E. 混合需求（多品类/多场景）
```
通勤西装+高跟鞋+通勤包，口碑好、划算？
户外徒步鞋+冲锋衣+双肩包，口碑好、划算？
搭配好看的围巾/帽子/腰带？
2025春季流行色、爆款连衣裙，卖得好的多推荐下？
```
### F. 其他特殊场景/问题
```
出差便携的鞋子 / 能塞进行李箱的？
进口品牌服饰口碑好的？
XX明星代言的衣服？
小红书/博主推荐过的？
有没有测评过的好穿牛仔裤？
有没有专柜同款/设计师联名款？
肤色偏黄，穿什么颜色显白？
160cm、120斤穿什么码？
好看的T恤/裤子？（用户错字修正）
不要假滑感的化纤面料衣服？
没更便宜的？
适合送女友/妈妈的礼物？
礼盒装/节日限定款有吗？
```
